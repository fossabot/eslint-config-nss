# eslint-config-nss

[![npm version](https://badge.fury.io/js/eslint-config-nss.svg)](http://badge.fury.io/js/eslint-config-nss)

## Install

```sh
yarn add --dev eslint-config-nss
```

## Usage
Then, add this to your .eslintrc file:

- Javascript
```json
{
  "extends": "nss"
}
```

- Node
```json
{
  "extends": "nss/node"
}
```

- React
```json
{
  "extends": "nss/react"
}
```
